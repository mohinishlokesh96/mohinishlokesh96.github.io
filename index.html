Alcohol Detection Meter

Description:

Alcohol Detection Meter is an Arduino(Arduino Uno) based project. This project helps solve one of the most commonly faced problem around the world. The detection meter helps in finding out whether the person driving the car is drunk or not. The application also pre-determines whether they are fit to drive or not. The project also helps in stopping the car when found the person to be drunk.

Attributes:

  • Implemented a kill switch device in the car using an Arduino(Arduino Uno) when the driver is detected to be drunk.
  • A camera is integrated to see rapid eye movements to check whether the person is drunk or not. Also MQ-3 sensor is used to check Blood Alcohol Content
  • Python and Java was used to build a simple game on the car to see whether the person is fit to drive.

Tools/Github Repository:

  • Tools: Arduino, Python, Java,Atom(IDE),MQ-3 Sensor.
  • Github Repository: Alcohol Detection Meter
